# 垂直水平对齐

`justify` `align` 等价于`justify-contents` `align-items`，用来设置 `Box` 的主轴方向、垂直主轴方向的对齐方式，即：

-   `direction` 为 `row` 时， `align` 控制垂直方向, `justify` 控制水平方向；
-   `direction` 为 `column` 时(默认值)， `justify` 控制垂直方向, `align` 控制水平方向。
