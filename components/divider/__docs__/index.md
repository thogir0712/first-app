# Divider

-   category: Components
-   family: General
-   chinese: 分隔符
-   type: 展示
-   version: 1.19

---

区隔内容的分割线, 支持版本 1.19 及以上。

## 何时使用

-   对不同章节的文本段落进行分割。
-   对行内文字/链接进行分割，例如表格的操作列。

## API

### Divider

| 参数        | 说明                 | 类型                          | 默认值   | 是否必填 |
| ----------- | -------------------- | ----------------------------- | -------- | -------- |
| dashed      | 是否为虚线           | boolean                       | false    |          |
| direction   | 线是水平还是垂直类型 | 'hoz' \| 'ver'                | 'hoz'    |          |
| orientation | 分割线标题的位置     | 'left' \| 'right' \| 'center' | 'center' |          |

## 无障碍键盘操作指南

分割线默认不会聚焦，无键盘操作。
